## 🎨 Decisión del Framework: CustomTkinter

He seleccionado **CustomTkinter** por las siguientes razones:

### Ventajas:
1. **Compatibilidad**: Basado en Tkinter, migración sencilla
2. **Moderno**: Aspecto visual contemporáneo
3. **Ligero**: Sin dependencias pesadas
4. **Mantenido**: Activamente desarrollado
5. **Documentado**: Documentación completa
6. **Flexible**: Temas, animaciones, widgets personalizados

### Por qué NO otras opciones:
- **PyQt5/PySide6**: Demasiado pesado, licencia compleja
- **Kivy**: Orientado a móviles, excesivo
- **wxPython**: API antigua, menos moderno
- **Dear PyGui**: Muy nuevo, menos estable
